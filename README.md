##  Como Utilizar

### Pré-requisitos

- [MySQL](https://dev.mysql.com/downloads/mysql/) instalado na sua máquina
- [MySQL Workbench](https://dev.mysql.com/downloads/workbench/) instalado
- Conhecimentos básicos em SQL (para os projetos introdutórios)

### Executando um projeto

1. **Clone o repositório**
```bash
   git clone https://github.com/cavalcantiraissas/MySQL-Workbench.git
```

2. **Abra o MySQL Workbench** e conecte-se à sua instância local do MySQL.

3. **Navegue até a pasta do projeto desejado** e abra o arquivo `.sql` correspondente.

4. **Execute o script** utilizando o atalho `Cmd + Shift + Enter` para rodar todo o arquivo, ou selecione trechos e pressione `Cmd + Enter` para execuções parciais.

5. Leia o `README.md` interno de cada projeto para entender o contexto, o modelo de dados utilizado e os objetivos das consultas.
